# Portfolio

> Personal portfolio website — cybersecurity analyst, full-stack developer, and AI builder.

## What it does

Single-page portfolio with sections for: About, Professional Experience, Featured Projects, Technical Skills, Education& Certifications, and Contact.

## Usage

Open `index.html` in any browser. No build step or server required.

## Features

- **Light/dark theme toggle**
- **Responsive** — works on mobile and desktop
- **Smooth scroll navigation**
- **Contact section** with social links
- **Zero dependencies** — pure HTML, CSS, and vanilla JS

## Files

```
portfolio/
├── index.html   # Main page
├── styles.css   # Styles
├── script.js    # Theme toggle + interactions
└── README.md    # This file
```

## Deploy

Host anywhere — GitHub Pages, Netlify, Vercel, or any static file host.
